Desafío #1: Sección de Comentarios
Crea una interfaz web que represente la sección de comentarios de una plataforma, con
las siguientes historias de usuario:
● Como usuario, me gustaría poder agregar un comentario, así puedo expresar las
● cosas que pienso.
● Como usuario, me gustaría ver una lista de comentarios anteriores, así puedo
revisar lo que escribir anteriormente.
● Como usuario, me gustaría poder eliminar un comentario, así puedo tener control
sobre lo que escribo.



Hacker edition:
● El sistema deberá utilizar GraphQL en el servidor con Node.js >=9.0 y consumir el
API generado utilizando Apollo Client 2.0.
● Los comentarios se deben de guardar en una base de datos como sqlite o mysql
utilizando Sequelize como ORM y el deploy con now.sh.
● Te aconsejamos revisar:
○ Construir servidor en GraphQL con Node.js
○ Integrar GraphQL y Apollo Client
